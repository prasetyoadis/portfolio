SIP Sistem Informasi Perpustakaan

Oleh Prasetyo Adi S. | date |

"Aplikasi web informasi untuk pengelolaan data perpustakaan secara efektif, efisien, terintegrasi, dan mudah diakses."

Ringkasan

SMA Negeri 5 Madiun merupakan salah satu sekolah menengah atas yang terdapat di kota Madiun. Di SMA ini terdapat sebuah perpustakaan yang melayani seluruh siswa-siswi, guru, maupun pengunjung luar sekolah. Pelayanan yang dilakukan oleh pustakawan masih konvensional sehingga menjadi tidak efektif dan efisien. Dengan demikian diperlukan teknologi komputasi berupa sistem informasi perpustakaan. Pengembangan sistem informasi perpustakaan berbasis web ini menggunakan metode pengembangan Waterfall serta menggunakan Framework PHP Laravel, dan Framework CSS Bootstrap.

Latar Belakang

Teknologi Informasi berkembang pesat dan sangat diperlukan di berbagai bidang, termasuk pendidikan. Penggunaan teknologi komputer membuat pekerjaan lebih efisien. Salah satu contohnya adalah sistem informasi berbasis web yang dapat meningkatkan pengelolaan data di perpustakaan. Sistem ini terbagi dua: untuk pustakawan yang mencakup pendataan buku, pendataan anggota serta transaksi peminjaman, dan untuk pengunjung yang mencakup presensi dan pencarian buku. SMA Negeri 5 Madiun masih menggunakan metode konvensional dalam pengelolaan data, sehingga penulis berencana merancang sistem informasi perpustakaan berbasis web untuk mempermudah proses ini.

Fitur Utama

  • 🔍 Pencarian Koleksi Buku Perpustakaan.
  • 📝 Pencatatan Kehadiran Pengunjung.
  • 📚 Kelola Koleksi Buku.
  • 👥 Kelola Anggota Perpustakaan.
  • 📝 Pencatatan Transaksi Peminjaman Buku.
  • 🧾 Pengembalian Buku.

Teknologi yang Digunakan

  • FrontEnd  : HTML, CSS (Bootstrap 5), JavaScript (jQuery)
  • BackEnd  : PHP (Laravel 8)
  • Database  : MySQL
